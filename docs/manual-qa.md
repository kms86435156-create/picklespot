# PBL.SYS 수동 검증 시나리오

> 베타 오픈 전 비개발자도 따라할 수 있는 검증 체크리스트입니다.
> 각 시나리오를 순서대로 실행하고, "정상 결과"와 일치하는지 확인하세요.

---

## 사전 준비

- 배포된 사이트 URL 확인 (예: `https://pbl-sys.vercel.app`)
- 관리자 계정 정보 확인 (ADMIN_USERNAME / ADMIN_PASSWORD)
- 테스트용 전화번호 1개 준비

---

## 시나리오 1: 홈 화면 진입

**단계:**
1. 브라우저에서 사이트 메인 URL 접속

**확인 항목:**
- [ ] 페이지가 에러 없이 로딩됨
- [ ] 상단에 "PBL.SYS" 로고와 네비게이션 메뉴가 보임
- [ ] 메뉴: 대회, 피클볼장, 동호회
- [ ] "추천 대회" 섹션에 대회 카드가 표시됨
- [ ] "지역별 피클볼장" 섹션에 지역 카운트가 표시됨
- [ ] "모집중인 동호회" 섹션에 동호회 카드가 표시됨
- [ ] 모바일에서도 정상 표시됨 (브라우저 크기 줄이거나 모바일 기기)

**정상 결과:** 모든 섹션이 데이터와 함께 정상 렌더링. 빈 섹션이 없음.

---

## 시나리오 2: 대회 상세 진입

**단계:**
1. 홈에서 추천 대회 중 하나 클릭
2. 대회 상세 페이지 확인

**확인 항목:**
- [ ] 대회명이 정상 표시됨
- [ ] 시작일/종료일이 표시됨
- [ ] 장소명이 표시됨 ("장소 확인중"이 아닌 실제 장소명)
- [ ] 참가비가 표시됨
- [ ] 종별(남복/여복/혼복 등)이 표시됨
- [ ] "참가 신청" 버튼이 보임
- [ ] 뒤로가기 시 대회 목록으로 정상 이동

**정상 결과:** 대회 정보가 빠짐없이 표시되고, 참가 신청 버튼이 동작 가능 상태.

---

## 시나리오 3: 참가 신청하기

**단계:**
1. 대회 상세에서 "참가 신청" 버튼 클릭
2. 신청 폼에 정보 입력:
   - 이름: 테스트참가자
   - 연락처: 010-0000-1234
   - 종별: 하나 선택
   - 개인정보 동의 체크
3. "신청하기" 클릭

**확인 항목:**
- [ ] 신청 폼이 팝업/모달로 열림
- [ ] 필수 항목 미입력 시 에러 메시지가 표시됨
- [ ] 모든 항목 입력 후 신청 성공 메시지가 표시됨
- [ ] 신청 후 폼이 닫힘

**정상 결과:** "신청이 완료되었습니다" (또는 유사 성공 메시지) 표시. 에러 없음.

---

## 시나리오 4: 중복 신청 확인

**단계:**
1. 같은 대회 상세에서 다시 "참가 신청" 클릭
2. **동일한 연락처** (010-0000-1234)로 다시 신청 시도

**확인 항목:**
- [ ] 중복 신청 거부 메시지가 표시됨
- [ ] "이미 신청되어 있습니다" 또는 유사한 안내 문구

**정상 결과:** 같은 연락처로 같은 대회에 두 번 신청이 불가능. 명확한 거부 메시지.

---

## 시나리오 5: 관리자 — 접수 승인

**단계:**
1. `/admin` 접속 → Basic Auth 로그인
2. 좌측 메뉴에서 "대회 접수 관리" 클릭
3. 시나리오 3에서 신청한 건 확인
4. "승인" 버튼 클릭

**확인 항목:**
- [ ] Basic Auth 팝업이 정상 표시됨
- [ ] 로그인 후 관리자 대시보드 접근 가능
- [ ] 대시보드 상단에 스토리지 상태 배너가 표시됨 (데모모드/실운영모드)
- [ ] 접수 관리 페이지에서 신청 건이 보임
- [ ] 신청자 이름, 연락처, 종별이 표시됨
- [ ] 승인 후 상태가 "승인됨"으로 변경됨

**정상 결과:** 신청 건이 목록에 보이고, 승인 처리 후 상태 변경 확인.

---

## 시나리오 6: 운영자 리드 폼 제출

**단계:**
1. `/for-clubs` 페이지 접속
2. 운영자/동호회 대표용 문의 폼 작성:
   - 담당자명: 테스트운영자
   - 동호회명: 테스트동호회
   - 연락처: 010-0000-5678
3. 제출

**확인 항목:**
- [ ] 페이지가 정상 렌더링됨
- [ ] 폼 제출 후 성공 메시지가 표시됨
- [ ] `/admin/leads`에서 해당 리드 확인 가능

**정상 결과:** 리드가 관리자 페이지에 "신규" 상태로 등록됨.

---

## 시나리오 7: CSV 업로드

**단계:**
1. `/admin/tournaments` 접속
2. "CSV 샘플 다운로드" 클릭 → 파일 저장
3. 엑셀에서 열고 테스트 대회 2~3건 입력
4. CSV UTF-8 형식으로 저장
5. 관리 페이지의 CSV 업로드 영역에 드래그앤드롭

**확인 항목:**
- [ ] CSV 샘플 다운로드가 정상 동작
- [ ] 업로드 후 성공/오류 건수가 표시됨
- [ ] 업로드된 대회가 목록에 바로 반영됨
- [ ] 사이트 메인에서도 해당 대회가 보임

**정상 결과:** 업로드 성공 건수 표시. 목록과 사이트에 즉시 반영.

---

## 시나리오 8: Guides 공개 확인

**단계:**
1. `/guides/tournament-guide` 접속
2. `/guides/venue-guide` 접속
3. `/guides/organizer-guide` 접속

**확인 항목:**
- [ ] 각 가이드 페이지가 에러 없이 렌더링됨
- [ ] 제목과 본문 내용이 표시됨
- [ ] 과장되거나 현재 없는 기능을 약속하는 문구가 없음
- [ ] 모바일에서도 정상 표시됨

**정상 결과:** 3개 가이드 모두 정상 렌더링. 내용이 실제 기능 범위와 일치.
